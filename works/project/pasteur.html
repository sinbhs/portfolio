UI / UX design

  • 기존 쇼핑몰은 2013년에 만들어져 노후화된 디자인과 작은 글씨크기로 홈페이지 이용에 어려움이 많았음
    이를 해결하기 위해 가독성 좋은 깔끔한 폰트와 풀스크린 레이아웃, 적당한 여백을 배치하여
  • 기존 웹사이트에서 부족했던 심미적, 기능적 부분 개선
  • PC 화면에서 사용자가 동작 상태를 인지할 수 있도록 CSS를 이용한 마우스 이벤트 발생

publishing

  • W3C Validator 검사 완료
  • 웹표준, 접근성을 고려한 HTML 마크업 언어 구축
  • mobile(375x812), pc(1920x1050) 스크린 사이즈에 최적화
  • 화면 크기에 따라 반응하는 반응형 웹 구현
  • 동적인 요소를 위해 javascript, AOS 플러그인 사용
